# Contributing

## Dev quickstart
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt -r requirements-dev.txt
pytest -q   # pytest >= 8 is expected
```

## Optional stacks
Install heavy deps only when needed:
```bash
pip install -r requirements-extra.txt
```
